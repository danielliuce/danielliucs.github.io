May:

I started off this internship without any knowledge in Python, Machine Learning, or Federated Learning. I quickly learned Python and the basic concepts of federated learning within my first two days of the internship.

For the next two weeks, I took a deeper dive into learning the concepts of reinforcement learning from the book: Deep Reinforcement Learning Hands-on by Maxim Lapan. I studied only select chapters related to what would be present during this internship. Specifically chapters 1->7, 9->11. The hands on code that I wrote for each chapter from the book can be found on my public repositories on my github.

The third week was when I started running experiments for a paper that was worked on before I arrived for this internship. I also edited existing plots for previous experiments in this paper as well as write code to find the average reward at every 10k timestep of these experiments. This paper propsed an algorithm Lethe that was for continual reinforcement learning which aimed to forget previous tasks that interfered with the current task so that agents can adapt and converge quicker to newer tasks. It used a prioritized replay buffer based on an interference metric that encourages the agent to focus on samples from the new task when the gradients of previous tasks conflict with the new one.

After Lethe was submitted, the fourth week was when I started to work on extending the federated learning framework Plato
